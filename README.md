<p dir="auto">You can use the attached code simply to compare it to yours. But you can also run it.</p>
<p dir="auto">To run my code, navigate into a specific code snapshot folder via the <code>cd</code> command in your command prompt or terminal first.</p>
<p dir="auto">Then run <code>npm install</code> to install all required dependencies (this will create a <code>/node_modules</code> folder).</p>
<p dir="auto"><strong>Important:</strong> If you're using the code for a module that requires API keys or a backend (e.g. the module about sending Http requests), you'll have to use <strong>your backend URLs</strong> or API keys. Mine won't work (I disabled my projects).</p>
